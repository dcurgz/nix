This is a NixOS flake for a variety of systems managed by @dcurgz.

It's written with `flake-parts` and `import-tree` which together allow a
modular approach to declaring a NixOS, nix-darwin, and home-manager
configurations.

I used to use Agentic AI (like Claude) to manage this, and it was helpful with
learning the basics of Nix and NixOS, but nowadays I write all of my code by
hand. (I know, crazy right?)
